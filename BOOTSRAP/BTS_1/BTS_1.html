Cos'è bootstrap

1. Bootsrtap è un framework front-end per poter sviluupare in modo semplice e più veloce un sito web.

Cosa devi importare per creare una pagina web con bootstrap

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

cosa è un CDN

Una CDN è un network di server distribuiti in diverse aree del mondo, che mantengono una copia del tuo sito e dei suoi file statici la CDN aumenta la velocità di caricamento delle pagine. La cdn rispetto a scaricare boostrap sul mio sito ha un vantaggio,che è quello di pesare di meno rispetto a bootstrap

A cosa serve questo attributo del tag meta?

Che differenza c’è tra Container e Container Fluid

La containerclasse fornisce un contenitore responsive a larghezza fissa La container-fluidclasse fornisce un contenitore a larghezza intera , che copre l'intera larghezza della finestra
